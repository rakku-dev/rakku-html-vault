FAQ

Pertanyaan yang sering ditanyakan

Semua info penting disatukan biar kamu nggak bingung.

  • Premium adalah membership bot untuk pribadi (Bukan untuk grup).
  • Premium memberikan akses hampir semua fitur dan limit unlimited.
  • Premium Permanen berlaku selama bot masih beroperasi.
  • Premium tidak akan hilang meski bot berganti nomor.
  • Premium akan hilang jika kamu mengganti nomor WhatsApp kamu.
  • Premium tidak dapat dipindahkan ke nomor lain.
  • Premium tidak dapat direfund.
  • Membership bot untuk grup; semua anggota grup dapat menggunakan bot dalam grup terdaftar.
  • VIP memiliki batasan penggunaan command per hari untuk setiap anggota.
  • VVIP tidak memiliki limit harian (Unlimited Limit).
  • VIP/VVIP tidak dapat dipindahkan ke grup lain.
  • Dilarang menyewa untuk grup terkait LGBT, Judi Online, dan praktik ilegal lainnya.
  • Kamu harus menyiapkan nomor WhatsApp sendiri untuk dijadikan bot.
  • Admin tidak menyediakan nomor bot pada membership Jadibot.
  • Jadibot Exclusive memiliki server lebih optimal dan responsif dibanding Reguler.
  • Admin tidak bertanggung jawab jika nomor kamu terbanned oleh WhatsApp.
  • Limit direset setiap hari pukul 01.00 WITA (UTC+8).
  • Batas limit harian mengikuti level (semakin tinggi semakin besar).
  • Untuk unlimited limit gunakan Premium dengan command !premium.
  • Naikkan level dengan aktif menggunakan bot.
  • Lihat level & profil via !info.
  • Dapatkan coin via !claim/!redeem & pantau saluran Rakku Waiter untuk kode redeem.
  • Kode redeem: saluran WA rakku.cloud/channel & YouTube @kyourakku.
  • Coin: membeli assistant/AI mode di !shop.
  • Assistant: mengganti kepribadian AI Chat sesuai preferensi.
  • Credit: membeli premium, konversi ke coin, & akses Image Generation. Beli via !Admin.
  • Tidak ada fitur bug WA.
  • Tidak ada fitur push kontak (mengganggu kenyamanan publik).
  • Tidak menjual panel bot.
  • Club pada profil hanya untuk pajangan.
  • Jadibot free dapat mengirim pesan promosi: hilangkan dengan Premium atau VIP/VVIP.
  • Secara default Jadibot Reguler merespon semua chat yang match command. Gunakan banchat (lihat !MenuJadibot) untuk membatasi.
  • Admin Jadibot tidak dapat diganti.
